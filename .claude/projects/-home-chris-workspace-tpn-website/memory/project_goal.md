Rebuilding https://texasphilanthropynetwork.com/ using Next.js. The current site is being replaced with a more polished, professional design.

**Why:** TPN is a non-profit organization — the site needs to convey professionalism and elegance.

**How to apply:** Design choices should lean toward clean, sophisticated aesthetics appropriate for a nonprofit. Prioritize readability, trust signals, and a polished feel over flashy/trendy design.
